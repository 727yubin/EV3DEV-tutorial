1. The Basics

What is Python? Python is a programming language. A program is just a set of instructions that you give to a computer to follow. It is almost like a recipe. There are loads of programming languages. Some are old, such as C and C++. Others are new, like Rust.

Why Python? Python is a high-level programming language, which means you do not need to give the computer ultra-specific instructions. Don’t worry if you didn’t understand this bit.

First, search for “Terminal” in Applications. Open it and type

$ python3

By the way, that dollar sign is just to tell you that you are in the terminal. Don’t type the dollar, just type python3. No capitals, computers are very picky.

And a word of advice, do not copy and paste from this page into the terminal.

Then you’ll see something like this.

$ python3 Python 3.6.2 (v3.6.2:5fd33b5, Jul 8 2017, 04:57:36) [MSC v.1900 64 bit (AMD64)] on win32 Type "help", "copyright", "credits" or "license" for more information. >>>

And now you’re in the Python 3 interpreter. What you see on the first line in parentheses will be different, but do not worry.

If you see something like Python 2.7, you are in Python 2, which is not what you should be in. You probably typed python, not python3.

Let’s start off with some arithmetic, shall we?

>>> 2 + 2 4 >>> 2+2 4 >>> 2 + 2 4

Throughout this tutorial, what you see after >>> is what you type in Python 3. What you see after $ is what you type in the terminal. If there is nothing in front, that is the computer.

Hmm. Having any number of spaces between a number and the plus sign doesn’t matter. It is good practice, though, to put one space between a number and a sign.

Let’s try some other things.

>>> 5 - 2 3 >>> 2 * 3 6 >>> 7 / 3 2.3333333333333335

Okay… what just happened? Computers use floating points, so the final few digits will not always be correct. Don’t worry about it at this stage.

Let’s do something more advanced.

>>> 7 % 2 1 >>> 3 ** 2 9

% is the modulo operator. It gives the remainder of the number after dividing it. 7 / 2 = 3 R1.

** is raising a number to a power. 3 to the power of 2 is 9.

>>> 2 + 5 * 3 17

Python follows the order of operations. (PEDMAS, or BIDMAS for the GCSE M0 lads out there. Parentheses and exponents are the same things as brackets and indices.) Keep this in mind.

In cases like this, it is more preferable to write the above as: >>> 2 + 5*3 17 to easily read the order of operations.
