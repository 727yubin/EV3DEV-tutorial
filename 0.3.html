Guide/Advice

  1. Work at your own pace. Some people understand programming very quickly, but others take time. It’s fine to cover one section every 3 weeks or cover 5 sections in a few hours, as long as you can grasp the idea.
  2. Feel free to go back at any stage.
  3. Don’t rush. Make sure you grasp each idea firmly, but don't aim for perfection. It's ok if you don't understand a small part.
  4. Whenever you see an example, try it out for yourself.
  5. Ask for help from others. Often, it’s much easier to understand something when someone next to you explains it.

Any updates can be found at https://github.com/727yubin/EV3DEV-tutorial.
