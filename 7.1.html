7. Loops

So far, we’ve only been able to do things that just go in a series. What if we want to repeat something?

For that we use loops. Let's first look at a while loop.

This is the format:

while condition: do_something

A while loop is just like an if statement, except that it repeats whatever is inside the loop until the condition is not met.

Let’s start with something simple. Save the following as while_loop.py.

1 #!/usr/bin/python 2 3 x = 2 4 5 while x <= 10: 6 print(x) 7 x = x + 1 8 9 print(“Done”)

As a side note, it is good practice to put a line between chunks of code, like what is done above. It makes no difference to the computer, but it is more pleasing to the eye.

Now run the program:

$ python3 while_loop.py 2 3 4 5 6 7 8 9 10 Done

Let’s go through what exactly happened. In line 3, we set the variable x to be 2. In line 5, the computer checks if the value of x (in this case, 2) is less than or equal to 10. <= means less than or equal to. Since x is less than 10, the program runs whatever is in lines 6 and 7; line 6 printing the value of x (2), and line 7 increases the value of x by 1. The program then goes back to the top of the loop.

x is now 3. 3 is less than 10, so the computer goes through lines 6 and 7 again, until x is 11. 11 is greater than 10, so it skips the block inside the while loop, and goes to line 9, and prints Done.

Can you predict what would happen if you indent line 9?
