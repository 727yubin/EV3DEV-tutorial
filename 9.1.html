9. Some Useful Modules

There are things in python called modules. These modules give python additional functionality. To use them in your program, you will have to put import module_name at the beginning of your program.

The first module is called time. This is probably the most useful module. A basic use of this module is making a motor turn for 5 seconds.

>>> import time >>> time.time() 1532635027.0614598 >>> time.sleep(1) >>>

When you run time.sleep(1), you’ll notice that the >>> doesn’t appear for precisely one second.

What does time.time() do? It gives the Unix timestamp, which is the number of seconds from January 1st, 1970. An example of this would be a while loop that only runs for 2 seconds. Line numbers will not be written from now on. #!/usr/bin/python import time start_time = time.time() while time.time() < start_time + 2: print(“Time not over”)

Before the while loop starts, we store the Unix timestamp in the variable start_time. In the loop, we check if the Unix timestamp is greater than start_time plus 2 seconds.

The second module is sys. You will probably only use sys.exit() which is just a fancy break.

The third module is random. It does things related to randomness(or pseudo-randomness).

>>> from random import randint >>> randint(1, 10) 8 >>> randint(1, 10) 10 >>> randint(1, 10) 1 >>> randint(1, 10) 3

>>> from random import choice >>> list = [2, 3, 4, 5, 6, 7, 8] >>> choice(list) 3 >>> choice(list) 2 >>> choice(list) 6 >>> choice(list) 2 >>> choice(list) 6
