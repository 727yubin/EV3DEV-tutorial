Logic Operators

The second method is using logic operators.

1 #!/usr/bin/python 2 first_name = input(“What is your first name? “) 3 last_name = input(“What is your last name? ”) 4 if first_name == “John” and last_name == “Doe”: 5 print(“Welcome, John Doe! ”) 6 else: 7 print(“Access denied. ”)

Notice the and in line 4. and is one of four logic operators, the other three being or, not, and xor. Let’s see what each of these do.

>>> True and True True >>> True and False False >>> False and True False >>> False and False False

So an and will give a True only when both inputs are true.

>>> True or True True >>> True or False True >>> False or True True >>> False or False False

An or gives a True if any one of its inputs is true. >>> not True False >>> not False True

A not “flips” the input. >>> True ^ True False >>> True ^ False True >>> False ^ True True >>> False ^ False False

This character, ^, is Shift + 6. This is what is used in python to denote xor, which gives True only when one if its inputs is true. It is probably the least used out of the 4, but it is good to know it in case you come across it.
