Sensors

Now let’s look at sensors. Sensors go into the number ports. In your EV3 set, there are the following sensors(in subjective order of use, starts from most used):

Infrared I = ev3.InfraredSensor() Ultrasonic U = ev3.UltrasonicSensor() Color C = ev3.ColorSensor() Light L = ev3.LightSensor() Touch T = ev3.TouchSensor() Sound S = ev3.SoundSensor()

For the purposes of this tutorial, these are the variable names. You do not need to specify ports, unless you have 2 of the same sensors plugged in.

Infrared

You first have to set the mode of the sensor. Probably the only two you will use are 'IR-PROX' and 'IR-REMOTE'. You may need ‘IR-SEEK’ in some cases.

So first set the mode by doing I.mode = “IR_PROX”

You can get the distance by doing I.value()

For I.mode = “IR-REMOTE”

you can do: I.beacon(channel = <number>) I.bottom_left() I.bottom_right() I.top_left() I.top_right() which gives a boolean depending on whether the specific buttons is pressed. The beacon button is the button on the top. Note that the channel number starts from 1, not 0.

In I.mode = “IR-SEEK”

you can use I.distance(channel = <number>) I.heading() to find the distance and direction that the IR remote is in.

Ultrasonic

The only mode you probably will use is

U.mode = “US-DIST-CM”

U.value() will give the distance to the object in front of it in millimeters, not centimeters.

Color

There are 3 modes.

C.mode = “COL-AMBIENT” This is used to get the ambient light(light in a room).

C.ambient_light_intensity gives the ambient light.

C.mode = “COL-COLOR” This is used to get the color of what the color sensor is reading.

For individual values you can do C.red C.green C.blue

To get all the values at once, do C.raw

C.mode = “COL-REFLECT” This is used to read the amount of light that is reflecting off the object that the color sensor is facing.

C.reflected_light_intensity gives the reflected light.

Light

The light sensor has 2 modes that are very similar to the color sensor.

L.mode = “AMBIENT” L.ambient_light_intensity gives the ambient light.

L.mode = “REFLECT” L.reflected_light_intensity gives the reflected light.

Touch

T.is_pressed will give a boolean. True if pressed, False if not pressed.

Sound

This will probably be the sensor that you will never use(I have never used it). This is from the documentation.

S.mode = 'DB' S.sound_pressure A measurement of the measured sound pressure level, as a percent. Uses a flat weighting.

S.mode = 'DBA' S.sound_pressure_low A measurement of the measured sound pressure level, as a percent. Uses A-weighting, which focuses on levels up to 55 dB.
