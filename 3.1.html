3. String Manipulation

Say we have the following:

>>> hello = “Hi guys!”

How would I get the 1st character? The 5th? The last? What if I want to split the string into 2 words?

Let’s see.

>>> hello[0] ‘H’

How does that work? Let’s take a closer look. I’ll write down the string, and how we would usually count the characters:

H i g u y s ! 1 2 3 4 5 6 7 8

Okay, so H is the 1st character. But why is it hello[0]? Because computers count from zero. So this is the computer’s perspective:

H i g u y s ! 0 1 2 3 4 5 6 7

Let’s test it.

>>> hello[1] ‘i’ >>> hello[2] ‘ ‘

If we want to start from the back, we can do:

>>> hello[-1] ‘!’ >>> hello[-2] ‘s’

So:

H i g u y s ! -8 -7 -6 -5 -4 -3 -2 -1

Let’s see multiple characters. I’ll keep the letter count for reference.

H i g u y s ! 0 1 2 3 4 5 6 7

>>> hello[:3] ‘Hi ‘ >>> hello[2:6] ' guy' So the first number is inclusive, and the second number is exclusive; notice how [2:6] gave the characters from 2 to 5.

Now let’s try typing in some other things into the terminal. You don’t have to type what is after the #. Go back to Ch. 1 if you do not understand why.

>>> len(hello) # length 8 >>> hello.lower() # all lowercase 'hi guys!' >>> hello.upper() # all uppercase 'HI GUYS!' >>> hello.swapcase() # change case 'hI GUYS!' >>> hello.title() # Capitalize first letter of every word 'Hi Guys!' >>> hello.split() # Split the string by spaces ['Hi', 'guys!']

These should be self-explanatory.
