4. input()

Until now, we haven’t asked the user for anything. Now, let’s ask the user something. Open gedit, write the following program, and save it as input.py:

1 #!/usr/bin/python3 2 name = input(“What is your name? ”) 3 print(“Your name is”, name)

Now run it.

$ python3 input.py What is your name?

I’ll type John and press Enter.

$ python3 input.py What is your name? John Your name is John

Now we can take inputs from the user!

Note: If you put the following in line 3:

3 print(“Your name is”, name, “!”)

You will see that it gives the following:

Your name is John !

To get rid of that space before the exclamation mark, you can either do:

3 print(“Your name is ” + name + “!”)

or

3 print(“Your name is ”, name, “!”, sep = ““)

Both give Your name is John! Notice that either way, you have to manually add in the space after Your name is.
