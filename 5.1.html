Conditions

Okay, but until now our programs have done the same things over and over, with just a few differences based on variables. What if we want to completely change behavior depending on something?

Enter the if statement. This is how it is written:

if condition: do_something elif condition: do_something_else else: do_a_third_thing

That is 4 spaces before each do_something. There can be any number of elifs, including zero, but an if statement must start with an if. It is preferable to add an else in case all else fails, especially in robots.

Let’s try something simple. Open gedit and save the following as if_statement.py.

#!/usr/bin/python 2 name = input(“What is your name? “) 3 if name == “Bob”: 4 print(“Welcome, Bob!”) 5 print(“You are now logged in. ”) 6 elif name == “John”: 7 print(“Welcome, John!”) 8 else: 9 print(“Access denied.”)

Let’s go through this, line by line. Line 2 asks the user for his or her name, and stores it in the variable name. In line 3, the program checks if name is Bob. Notice that you use two equal signs for checking. If name is Bob, it runs whatever is under it that is indented(has 4 spaces in front of it). In this case, it prints Welcome, Bob! then You are now logged in.

If name is not Bob, the program skips to line 6. elif is short for else if. The program checks if name is John. If it is, it prints Welcome, John!

If name is neither Bob nor John, the program goes to line 8, which then goes on to line 9, and prints Access denied.

Let’s run it:

$ python3 if_statement.py What is your name? Bob Welcome, Bob! You are now logged in. $ python3 if_statement.py What is your name? John Welcome, John! $ python3 if_statement.py What is your name? Alice Access denied.

Checking for numbers is similar to math: use ==.
