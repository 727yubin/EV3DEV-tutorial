for

Another kind of loop is a for loop. A for loop looks like this:

for var1 in var2: do_something

Let’s start by doing something simple.

>>> for i in range(4): ... print("Hello") ... Hello Hello Hello Hello

range(number) repeats whatever is in the for loop that many number of times.

Let’s see another example.

>>> sentence = "This is my second for loop." >>> for words in sentence.split(): ... print(words) ... This is my second for loop.

Remember that split() splits a string by spaces. The for loop goes over each split string, storing it in the variable words. Then the program prints words. for loops are very handy for robots, as it can go through a series of items or events.
