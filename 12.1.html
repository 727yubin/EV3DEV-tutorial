Connecting the brick

If you’ve made it to here, congratulations! Now let’s see how to control the robots using python.

First, you have to run robot programs in the brick. To do that, connect your brick to the computer using the USB cable. Then open a terminal and type the following:

$ ssh robot@ev3-<brick_number>.local

where <brick_number> is the number written on your brick. (1/2/3/boss)

You’ll be prompted to log in.

robot@ev3-<brick_number>.local’s password:

When you log in, you’ll see something like this:

TODO: add ascii art from ssh

So that you can access your brick from files, add sftp://robot@ev3-<brick_number>.local.

To change tabs to 4 spaces, do expand -t 4 filename.py
