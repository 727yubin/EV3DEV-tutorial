Nesting

Now, what if we want to check for two things? There are two ways.

The first is called nesting. Nesting means putting an if statement inside another if statement.

Open if_statement.py and change it to the following:

1 #!/usr/bin/python 2 first_name = input(“What is your first name? “) 3 last_name = input(“What is your last name? ”) 4 if first_name == “John”: 5 if last_name == “Doe”: 6 print(“Welcome, John Doe! ”) 7 else: 8 print(“Hello, John.”) 9 else: 10 print(“Access denied. ”)

In line 4, the program checks if the user’s first name is John. If it is, the program checks if the user’s last name is Doe in line 5. If it is, the program prints Welcome, John Doe! If the user’s first name is John but his last name isn’t Doe, the program prints Hello, John.

If the user’s first name is not John, we directly go to line 9 and print Access denied.

Nesting can be very useful if you want to check something else only if some condition is met, and skip the whole block of code if the condition is not met.
